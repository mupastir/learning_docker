# Swarm mode and microservices

**Terminology**
- A cluster = A swarm

Engines in a swarm run in **swarm mode**

**Manager nodes** maintain the swarm (
	- H/A - recommended 3 or 5
	- Only one is **leader**
)

**Worker nodes** execute tasks

**Services**
	- Declarative & scalable

**Tasks**
	- Assigned to workers
	- <-> containers
